# MateCare — Repositorio completo

## Estructura

```
01_documentacion/   Todos los documentos del proyecto (leer en orden numérico)
02_codigo_base/     Código base listo para subir a GitHub
03_diseno/          Paleta de colores, wireframes, referencias visuales
04_referencias/     Análisis técnico y guías generadas durante el proceso
```

## Orden de lectura recomendado

1. MateCare_01_arquitectura_tecnica.md   — visión general del sistema
2. MateCare_03_cuentas_y_servicios.md   — qué cuentas abrir y en qué orden
3. MateCare_05_configuracion_entornos.md — cómo configurar todo antes de codear
4. MateCare_04_guia_local_y_deploy.md   — desarrollo local y deploy a Railway
5. MateCare_02_pagos_IA_resiliente.md   — Stripe y sistema de fallback IA
6. MateCare_06_biblioteca_prompts.md    — contenido del prompt engine

## Stack
Mobile: React Native + Expo  
Backend: Node.js + Express + Prisma  
BD: PostgreSQL (Supabase)  
IA desarrollo: Gemini (gratis) → agregar Anthropic y OpenAI después  
Pagos: Stripe
